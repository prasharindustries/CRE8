# CRE8 – Blockchain Creator Economy Platform

##  Overview
CRE8 is a decentralized creator payment platform built on Polygon that enables instant, transparent revenue sharing using smart contracts.

##  Problem
- Delayed payments (30–60 days)
- High platform fees (30–50%)
- No transparency

##  Solution
- Smart contract-based payments
- Instant 80/20 revenue split
- No intermediaries

##  Tech Stack
- Solidity
- Hardhat
- Polygon (Amoy)
- Ethers.js
- HTML/CSS/JS

##  Features
- MetaMask integration
- Automated revenue split
- On-chain transparency

## Architecture
User → Frontend → Ethers.js → Smart Contract → Polygon
