Reimagining Creator Payments

CRE8 is a blockchain-based system designed to automate revenue distribution in the creator economy. It addresses inefficiencies such as delayed payments, lack of transparency, and dependency on centralized platforms.

Purpose

The primary purpose of CRE8 is to introduce a decentralized financial layer for creators. By leveraging blockchain and smart contracts, CRE8 ensures instant, transparent, and trustless payment distribution.

  • Eliminate intermediaries
  • Enable automated revenue splitting
  • Reduce payment delays
  • Increase trust and transparency

About CRE8

CRE8 is a decentralized application built using Solidity smart contracts deployed on the Polygon blockchain.

In the current creator economy, platforms like YouTube and Instagram generate revenue but do not provide automated distribution. Payments are received by a single entity and then manually distributed, leading to inefficiencies.

CRE8 solves this by introducing a smart contract-based payment system where funds are automatically split between stakeholders. The current implementation demonstrates an 80/20 revenue split between a creator and collaborator.

The system uses MetaMask for wallet interaction and Ethers.js for blockchain communication. The frontend is built using HTML, CSS, and JavaScript and hosted on GitHub Pages.

This project serves as a proof-of-concept for decentralized financial infrastructure in the creator economy.
